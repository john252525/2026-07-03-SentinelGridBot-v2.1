1.  **Получить API-ключи именно от Testnet.**
    Ссылка: `testnet.bybit.com`. Ключи от реального счета не подойдут и опасны. В настройках ключа на бирже **обязательно** сними галочку «Разрешить вывод средств». Оставь только «Торговля» и «Чтение».

2.  **Проверить лимиты символа.**
    Убедись, что на тестовой сети торгуется контракт `XAUUSDT` и что минимальный размер ордера, который ты корректируешь в коде, проходит по лотности. Бывает, что при сильном уменьшении размера (например, до $2) количество контрактов становится меньше биржевого минимума, и API вернет ошибку.

3.  **Запустить в Dry-Run (визуальный контроль).**
    Первый час после запуска не отходи от терминала биржи. Смотри, как появляются лимитные ордера. Дождись срабатывания хотя бы одного колена. Убедись, что в логах Python нет красных строк с ошибками API.
